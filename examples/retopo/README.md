**Honest note on the quad target:** Instant Meshes treats
`retopo_target_vertex_count` as a *hint*, not a contract â€” with
`pure_quad=True` + crease constraints the monument-valley mesh came out at
~4Ă— the requested 3000 verts (pure-quad extraction subdivides). Use
`decimate` when you need a hard budget; use `quad` when you need clean quad
flow for sculpting/subdiv and let the count land where the orientation field
puts it.

Deps: `pip install pyinstantmeshes` (quad), `pip install fast-simplification`
(decimate), `pip install trimesh` (smooth). Each path fails with an
actionable ImportError when its dep is absent; the workflows still load.
